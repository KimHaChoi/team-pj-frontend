# 📸 찰칵 (Chalkak) - 위치 기반 시민 환경·에너지 자율 관제 포털

> **대구광역시 달성군 기후 에너지 회복 연대 특별 시범 지구 가동**  
> **React + TypeScript + Tailwind CSS + Google Gemini AI + Firebase Dual-Active System**

## 🧭 1. 프로젝트 개요

**"찰칵(Chalkak)"**은 시민들이 도심 속 환경오염 정황, 에너지 낭비 사각지대, 무단 쓰레기 투기 등을 단 1초 만에 사진 촬영 및 자동 GPS 위치 매핑으로 신속하게 신고하고, 지자체 공무원과 연동된 스마트 SaaS 관제 시스템이 처리 상태를 실시간 보고 조치하는 **대국민 참여형 탄소 중립 에코 에너지 종합 포털**입니다.

### 🌟 기획 의도 및 사회적 가치
- **노휴먼스랜드(No-Human's Land) 폐지 이후 복원 가속화**: 도시 재건 과정에서 일어날 수 있는 무분별한 전력 유출, 대기 오염 매연 방출을 시민들의 눈(눈동자 정찰단)으로 선제 차단합니다.
- **AI 환경 심사 자동화**: 대량의 공공 민원 업무를 사람이 다 읽기 전에 Google Gemini AI가 시급성 점수(0~100)와 장난성/스팸 여부를 1초 만에 실시간 기계 판독하여 지자체 조치 우선순위를 기민하게 추천합니다.
- **게이미피케이션 동기부여**: 해결 완료 판정 즉시 시민에게 AI 중요도 점수 비례 에코 마일리지 포인트를 실시간 정산해 주어, 모바일 포인트숍에서 친환경 기프티콘 및 대구 제로페이 지역 화폐로 무상 바코드 실물 교환을 지원합니다.

## 🛠️ 2. 기술 스택 및 아키텍처 우수성

- **Front-end**: React 18, Vite, TypeScript, Tailwind CSS
- **Design Aesthetic**: Premium Emerald Carbon Green & Soft Glassmorphism System
- **Cloud Backend**: Firebase v10 Authentication, Cloud Firestore (실시간 실감 서브스크립션)
- **Edge AI**: Google Gemini 1.5 Flash client-side REST Engine (Serverless API)
- **Fault-Tolerant Hybrid DB**:
  - `isFirebaseActive` 자율 체크 회로 내장.
  - 파이어베이스 접속이 어렵거나 네트워크 오프라인 상태일 시, 즉각 브라우저 **`LocalStorage Live Replica` 복제 저장소**로 1초 만에 실시간 매끄럽게 우회 연동(Seamless Fallback)하여, 가동 중단율 0%의 무결점 해커톤 발표 무대를 보장합니다.
- **Dual Map Engine (이중화 지도 엔진)**:
  - 카카오/구글 맵 스크립트 로드 실패 시, 대구 테크노폴리스 5대 지구(중구-동구-생태공원-에코타운-송배망)를 **대화형 SVG 벡터 맵**으로 구현.
  - 시민 제보 실제 위경도 좌표를 상대 좌표 변환 공식에 대입하여 지도 위에 정확히 마커 핀(Pin Marker)으로 시각화하고, glowing gradient 그라데이션을 통한 **실시간 온열 히트맵 레이어(Heatmap Overlay)**를 실현했습니다.

## ⚙️ 4. 주요 핵심 화면 기능 명세

### 📱 A. 시민 모바일 전용 서비스 (Mobile-First Dashboard)
- **실시간 제보지도 (Map)**: 5대 구역별 실시간 환경 종합 지수(`environmentScore`)를 산출 게이지로 보이고, 카테고리/상태별 다중 슬라이더 필터링 및 타오르는 실시간 온열 히트맵 레이어 장착.
- **찰칵 간편신고 (Report Form)**: 브라우저 Geolocation API 자동 현장 매핑, 파일 첨부 리더, 설명 미입력 시에도 자동 제목 조립, 그리고 제출 클릭 시 Google Gemini 실시간 탄소 가치 평가 피드백 모달 오버레이 제공.
- **나의 보관함 (My Archive)**: 내가 제출한 제보에 대해 "접수 ➔ 조치 중 ➔ 해결 완료" 3단계 실시간 반응형 타임라인 노출 및 공무원이 작성해 준 해결 일장 서면 메모와 "조치 전/후 증빙 이미지" 비교 뷰어 제공.
- **기여 랭킹 (Leaderboard)**: 개인정보 보호를 위한 이메일 부분 마스킹(`***@`) 및 전국 순위 / 달성군 순위 / 우리 동네 순위 유동 범위 필터 탑재.
- **포인트숍 (Store)**: 제로페이 상품권, 텀블러 에코 쿠폰, 공용 자전거 패스를 마일리지와 맞바꾸고, 구매 즉시 실시간 벡터 바코드가 생성되어 오프라인 실물 매장에서 바코드 스캐너 스캔이 가능하도록 구성.

### 💻 B. 지자체 공직자 관리 패널 (Enterprise SaaS Dashboard)
- **종합 KPI 대시보드**: 누적 수신량, 해결율, 미조치 위해 민원 수 및 달성군 미니 관제 정보판.
- **우선순위 관제 대장**: Gemini AI 점수가 가장 높은 '긴급 복원' 건부터 최상단에 자동 오름차순 배치.
- **행정 조치 결재 서명**: 조치 담당 공무원명 지정, 행정 보고 메모 작성, 및 조치 완료 사진을 로컬 프리뷰 업로드하면 상태가 `RESOLVED`로 승인되며 동시에 제보 시민의 계정으로 포인트가 실시간 보상됩니다.
- **스마트 자산 제어 콘솔**: 시민들이 대낮 가로등 켜짐(에너지 낭비)을 제보하면, 해당 구역 스마트 가로등 디바이스에 빨간색 경보("점검 필요 🚨")가 실시간으로 깜빡이며, 관리자가 원격 스위치를 꺼주면 자동으로 경보가 클리어되는 고품격 인터랙티브 하드웨어 결합 제어판입니다.
- **시민 평판 제재**: 허위 신고 남발 시민을 실시간 원터치 즉각 차단/해제할 수 있는 평판 관리 단속 대장.

## 🚀 5. 로컬 설치 및 실행 방법

### 환경 변수 세팅 (`.env`)
프로젝트 루트 폴더에 `.env` 파일을 생성하고 발급받으신 Gemini API 키를 기입해 주십시오. (미입력 시에도 인공지능 Heuristic fallback 우회 작동하여 안전합니다.)
```env
VITE_GEMINI_API_KEY=your_gemini_api_key_here
```

### 패키지 설치 및 실행
```bash
# 1. 의존성 패키지 설치 (Vite, Tailwind, Context API, Lucide-react etc)
npm install

# 2. 로컬 개발 서버 부트스트랩 기동
npm run dev
```
기동 후 터미널에 표시되는 로컬 브라우저 주소(`http://localhost:5173`)로 접속하시면 찰칵 포털의 에메랄드 테마를 즉시 만끽하실 수 있습니다.
